This is a tool which helps to reach goals.

You can write down goal, decompose it to several tasks (and sub-tasks), mark deadlines for each of them.
Tasks can have a priority and severity, and connection to the other task (has to be done after, has to be done before, etc).
You can invite other person to work on task together.
You can filter them: status - new, in progress, complete; percentage of completion; etc.
You can tune visibility of your goal.
Every goal has a criterias of completion.
(the simpliest version of jira atlassian)
